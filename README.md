# 사이다 낚시왕 퍼블리싱 프로토타입

## 파일 구조

```text
cider-fishing-publish/
├─ index.html
├─ css/
│  └─ style.css
├─ js/
│  └─ app.js
└─ assets/
```

## 포함 화면

1. 홈
2. 낚시 화면
3. 낚시 결과
4. 세계지도
5. 지역 상세
6. 미션
7. 도감
8. 물고기 상세
9. 낚싯대
10. 포인트 상점
11. 업적
12. 랭킹

## 이번 수정에서 추가된 기능

- 하단 GNB 화면 전환
- 알림 모달
- 전체 메뉴 모달
- 탭 활성화
- 낚시 성공 → 결과 화면 이동
- 미션 보상 수령 상태 변경
- 금융 퀴즈 팝업
- 세계지도 지역 클릭 / 잠금 지역 안내
- 도감 카드 클릭 → 물고기 상세 반영
- 낚싯대 선택 / 장착 상태 표시
- 낚싯대 강화
- 상점 상품 구매 토스트
- 낚시 결과 공유 / 복사
- 토스트 메시지

## 사용 방법

`index.html`을 브라우저에서 열면 바로 확인할 수 있습니다.
모바일 WebView 기준 390 x 844 해상도에 맞춰 제작했습니다.
