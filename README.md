# SBI 사이다 낚시왕

시안 기반 모바일 웹 게임 프로토타입입니다.

## 반영 사항
- 메인 화면 미끼 수량: x1
- 낚시 화면 입질 버튼 하단 문구: 터치!
- 홈 / 세계 지도 / 낚시 / 도감 / 미션 / 낚싯대 화면 WebP 에셋 반영
- 낚시 화면 입질 버튼 클릭 시 게이지 미니게임 동작
- 중앙 초록 영역 성공, 그 외 실패
